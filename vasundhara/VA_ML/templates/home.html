1. The user simply have to fill up a form that contains the details about the soil's nutrition.
2. This provided information will create a report about the quality of the soil and what kind of crops can be grown in that soil.
3. The website will have backend machine learning approach to generate the artificial intelligence for deriving the best optimal solution for the farmer.
4. The user can also enter the type of crops he/she wants to grow in that soil and according to that the user will get the suggestions about it.
